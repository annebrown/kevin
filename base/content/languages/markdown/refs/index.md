## Escaping

| Char       | Markdown  |
| :--------: | :-------- |
| &lt;       | `&lt;`    |
| &amp;      | `&amp;`   |
| space      | `&nbsp;`  |
| &copy;     | `&copy;`  |

## Comments

| Sample Descr                            |
| -:--------------------------- | :------------------------------- |
| `<!--- this is a comment -->` | HTML - also supports block style |
| `[comment]: # (Single Line)`  | Link                             |
| `[//]: # (Single Line)`       | Link                             |
| `[/]: # (Single Line)`        | Link - Most Widely Supported     |
| `[comment]: word`             | Link - Single word               |
