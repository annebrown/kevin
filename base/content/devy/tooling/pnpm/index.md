## Description

Better performance and more efficient deps mgmt than npm

## CLI

```bash
pnpm rm # | uninstall | un
pnpm add pkg-name
pnpm i # | install [<pkgs>]
pnpm prune # Remove unness pkgs
pnpm cache delete # [<pkgs>]
```
