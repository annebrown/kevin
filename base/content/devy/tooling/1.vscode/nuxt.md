## Autocomplete

Edit `@@/.vscode/settings.json` for IntelliSense to autocomplete Nuxt `active-class` and `inactive-class` classes:

```json
{
    "tailwindCSS.classAttributes": [
        "active-class",
        "inactive-class"
    ],
}
```
